This is a shortcut for

```js
var zip = new PizZip();
zip.load(data, options);
```

Please see the documentation of [load]({{site.baseurl}}/documentation/api_pizzip/load.html).

**Example**

```js
var zip = new PizZip(data, options);
// same as
var zip = PizZip(data, options);
```
