---
title: "new PizZip() or PizZip()"
layout: default
section: api
---

**Description** : Create a new PizZip instance.

**Arguments** : None

**Returns** : A new PizZip.

**Throws** : Nothing.

<!-- __Complexity__ : Object creation in **O(1)**. -->

**Example**

```js
var zip = new PizZip();
// same as
var zip = PizZip();
```
